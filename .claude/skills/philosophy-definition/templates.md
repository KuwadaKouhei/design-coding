# 思想ドキュメント テンプレート（Templates）

`philosophy-definition` スキルの Process ステップ4で使う出力テンプレート。`docs/philosophy/` 配下に
以下の3ファイルを作成する。各テンプレートの `<...>` をインタビュー結果で埋める。
**決定（What）と理由（Why）を必ずセットで残す。**

共通方針:
- 文体は日本語。見出し階層は維持する。
- 「採用しない」と決めたことも明記する（後から「なぜ採用しなかったか」を辿れるように）。
- 末尾に「更新履歴」を置き、見直し時に追記する。
